# SI base units

|               | Quantity                  | SI unit  |     |
| ------------- | ------------------------- | -------- | --- |
| $t$           | time                      | second   | s   |
| $l, r, x$     | length                    | metre    | m   |
| $m$           | mass                      | kilogram | kg  |
| $I, i$        | electric current          | ampere   | A   |
| $T$           | thermodynamic temperature | kelvin   | K   |
| $n$           | amount of substance       | mole     | mol |
| $I_{\rm {v}}$ | luminous intensity        | candela  | cd  |
